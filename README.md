# Phase 1 Project — Python 🐍

Student Result Calculator + Quiz Game built as part of BTech CSE Python Full Stack Roadmap.

## Features

### Student Result Calculator
- Enter marks for multiple students and subjects
- Automatically calculates total, average, grade and pass/fail status
- Shows class summary with topper and failed students

### Python Quiz Game
- 3 difficulty levels — Easy (30s), Medium (20s), Hard (10s)
- Live countdown timer per question
- 10 Python questions with explanations
- Score and performance summary at the end

## Technologies Used
- Python 3.14
- Core concepts: Variables, Loops, Functions, Lists, Dictionaries, OOP, try/except

## How to Run
```bash
python phase1_project_v2.py
```
